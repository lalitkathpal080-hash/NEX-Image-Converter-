Supported Image Formats

Convert images to and from all major formats used on the web, in print, and in app development:

PNG
Lossless quality with full transparency support. Best for logos, icons and UI graphics.
JPG / JPEG
Compressed photo format. Smallest file size for photos, social media and web images.
WebP
Modern Google web format. Up to 30% smaller than JPG with excellent quality.
GIF
Animated format. Upload multiple images to create looping animated GIFs.
BMP
Uncompressed Windows bitmap. Maximum fidelity, larger file sizes.
ICO
Windows icon format (32×32). Essential for website favicons and app icons.
TIFF
High quality archival format used in professional photography and printing.
AVIF
Next-generation AV1-based format. Better compression than WebP, great for web.
SVG
Scalable vector embed. Infinite resolution at tiny file sizes. Perfect for icons.
PDF
Portable document format. Instantly embed any image into a shareable PDF.
